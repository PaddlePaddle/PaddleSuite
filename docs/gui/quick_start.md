# 快速开始

## 简介
感谢使用PaddleX可视化客户端，通过本客户端，您可以实现图像分类、目标检测、实例分割和语义分割四大视觉任务模型的训练，裁剪及量化，以及模型在移动端/服务端的发布。

## 快速安装
请参考[安装方式文档](./download.md)下载安装PaddleX可视化客户端。
