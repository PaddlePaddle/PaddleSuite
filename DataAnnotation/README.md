### LabelMe
LabelMe是目前广泛使用的数据标注工具，您也可以在保证标注文件格式与PaddleX所支持格式进行匹配的基础，上选用其他标注工具。  
LabelMe GitHub地址：https://github.com/wkentaro/labelme
